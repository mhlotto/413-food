# Guides template

Create a new file in this folder, e.g., `late-night.md` or `gluten-free.md`.

Template:

```markdown
# Guide title
Date: 2024-10-15
Focus: What this guide covers (late-night eats, study-friendly cafes, etc.)

## Spots
- [Restaurant Name](../data/restaurants.yaml): short note on why it fits this guide
- ...

## Tips
- Parking or transit notes
- Peak times to avoid
```

Keep guides concise and link back to restaurants in `data/restaurants.yaml`.
